## Como Resolver Type Alias no Typechecker

### 1. Adicionar no Hoist (`mod.rs:173-216`)
No método `hoist()`, adicionar caso para `ASTDeclarationKind::Alias`:
- Criar um symbol para o nome do alias
- Obter o `TypeId` do tipo target usando `get_typeid_of_name()`
- Criar uma entrada no `type_names` mapeando o nome do alias ao TypeId do target

### 2. Adicionar no Resolve (`mod.rs:218-317`)
No método `resolve()`, adicionar caso para processar o alias se necessário (geralmente apenas registra a declaração).

### 3. Resolução de Tipos
O alias deve ser transparente - quando alguém usar o nome do alias, deve resolver para o tipo target. As funções existentes como `retrieve_information_of_type` já fazem isso buscando no `type_names`.
